## Current research

**Adversarial ML detection in autonomous systems** — GWU D.Eng. dissertation

Designing a layered Bayesian/Kalman filter architecture with Bronze/Silver/Gold trust tiers for detecting adversarial manipulation of sensor inputs in safety-critical systems. Proof-of-concept: GPS spoofing in autonomous drone platforms. Detection coverage validated against MITRE ATT&CK tactics. Architecture performs multi-signal anomaly fusion with dynamic trust calibration across detection tiers.

> Adjacent to OT/ICS security: how do you detect when a safety-critical system's sensor inputs have been tampered with, and how do you maintain operational trust under adversarial conditions?
